연락처 폼 구현하기

아래 요구사항에 맞게 HTML 폼을 구현하세요:

  • 이름 (필수 입력)
  • 이메일 (필수 입력)
  • 전화번호 (선택 사항)
  • 성별 (라디오 버튼: 남성, 여성, 기타)
  • 관심 분야 (체크박스: 개발, 디자인, 마케팅)
  • 국가 (드롭다운 선택: 대한민국, 미국, 일본, 중국, 기타)
  • 메시지 (텍스트 영역)
  • 제출 버튼

폼 데이터는 POST 방식으로 "/submit" 엔드포인트에 제출되도록 설정하세요.

필수 입력 필드는 * 표시와 required 속성을 사용하세요.
